# brunobailosolo

Diario personal de Bruno. Un archivo por día.

---

## Cómo escribir cada día

1. Entra a `diario/2026/04-abril/`
2. Abre el archivo de hoy (ej: `2026-04-08.md`)
3. Haz clic en el **✏️ lápiz** (arriba a la derecha)
4. Rellena lo que quieras
5. Scroll abajo → **Commit changes** → **Commit directly to main** → botón verde

> ¿Dudas? Mira `_info/como-usar.md`

---

```
diario/2026/04-abril/
  2026-04-07.md
  2026-04-08.md
  ...
  2026-04-30.md
```
