# 🎵 brunobailosolo

Repositorio personal de Bruno — diario de vida, proyectos y tracking semanal.

> Simple. Directo. Sin más.

---

## 📁 Estructura

```
brunobailosolo/
├── diario/
│   └── 2026/
│       └── YYYY-MM-DD.md     # Un archivo por día
├── semanas/
│   └── semana-XX-2026.md     # Cierre semanal
└── README.md
```

---

## 📋 Formato del diario

Cada día tiene:
- 😴 Sueño
- ⏰ Horario real (qué hice)
- 📊 Hábitos
- 💬 Reflexión corta

---

## ⚡ Inicio rápido

1. Clonar: `git clone https://github.com/alvarofernandezmota-tech/brunobailosolo`
2. Abrir la carpeta `diario/2026/`
3. Copiar la plantilla y rellenar
