# 🎵 brunobailosolo — Diario de Bruno

> Diario personal simple. Un día = un archivo. 5 minutos al día.

---

## 🚀 Cómo usar esto (sin instalar nada)

Todo se hace **desde esta misma página web**. No necesitas saber programar.

### Para escribir el diario cada día:

**Paso 1** — Ve a la carpeta `diario/2026/`

**Paso 2** — Haz clic en `+ Create new file` (botón verde arriba a la derecha)

**Paso 3** — En el nombre del archivo escribe: `2026-04-08.md` (con la fecha de hoy)

**Paso 4** — Copia y pega el contenido de `_PLANTILLA.md` y rellena los campos

**Paso 5** — Scroll abajo, clic en **“Commit changes”** → **“Commit directly to main”** → **“Commit changes”** (botón verde)

> ✅ Listo. Ya está guardado.

---

### Para editar un archivo ya existente:

1. Abre el archivo que quieres cambiar
2. Haz clic en el ✏️ **lápiz** (arriba a la derecha del texto)
3. Edita lo que necesites
4. Scroll abajo → **“Commit changes”** → **“Commit directly to main”** → verde
