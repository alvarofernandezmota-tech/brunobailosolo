# Cómo usar este diario desde GitHub

> Esto es solo información. No hace falta tocarlo.

---

## Abrir el día de hoy

1. Ve a `diario/2026/04-abril/`
2. Haz clic en el archivo con la fecha de hoy (ej: `2026-04-08.md`)
3. Haz clic en el **✏️ lápiz** (arriba a la derecha del texto)
4. Rellena los campos que quieras
5. Scroll abajo → **Commit changes** → **Commit directly to main** → botón verde ✅

---

## Qué rellenar

| Campo | Cómo |  
|-------|------|
| Sueño | Escribe la hora después del `\|` |
| Horario | Escribe qué hiciste en cada bloque |
| Ejercicio | Cambia `⬜` por `✅` o `❌` |
| Cervezas | Cambia el `0` por el número real |
| Notas | Escribe lo que quieras, sin estructura |
| Nota del día | Escribe el número antes de `/10` |
