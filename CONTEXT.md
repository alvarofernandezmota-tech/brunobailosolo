# CONTEXT — brunobailosolo

## ¿Qué es este repo?

Diario personal de Bruno. Simple, honesto, sin florituras.

## Filosofía

- **Un día = un archivo**
- **Rellenar tarda < 5 minutos**
- **Cierre semanal = 1 archivo más**

## Estructura de archivos

```
diario/2026/YYYY-MM-DD.md
semanas/semana-XX-2026.md
```

## Cómo usar

1. Al final del día, abrir el archivo del día
2. Rellenar los apartados a mano
3. Commit con mensaje: `diary: YYYY-MM-DD`

## Hábitos que trackea

- Ejercicio
- Agua
- Pantallas antes de dormir
- Música / práctica

## Nota sobre el formato

Las notas del día van de 1 a 10.
No hay sistema de tags ni etiquetas complejas.
Menos es más.
