# 🐍 Python — Aprende desde cero

Ejercicos estructurados para aprender Python paso a paso.
Cada archivo tiene dos partes: **código ya resuelto** con explicaciones y **tu turno** para practicar.

| Archivo | Concepto | Dificultad |

| `ejercicio_01_variables.py` | Variables y tipos de datos | ⭐ |
| `ejercicio_02_listas.py` | Listas | ⭐⭐ |
| `ejercicio_03_condiciones.py` | Condiciones if/elif/else | ⭐⭐ |

## ▶️ Cómo ejecutar (sin instalar nada)

1. Ve a **[colab.research.google.com](https://colab.research.google.com)**
2. Nueva libreta → copia el contenido del ejercicio
3. Dale a ▶️ y completa la sección **TU TURNO**

---

## 🗂️ Estructura de cada ejercicio

```
# ── PARTE RESUELTA ──
# Código completo con comentarios que explican cada línea.
# Léelo, ejecútalo, entiende qué hace.

# ── TU TURNO ────────
# El mismo tipo de código pero con ___ para que lo completes tú.
```
